# Steam OpenID provider for Luvit

* [Example](examples/server.lua) (using [mooncake](https://github.com/cyrilis/luvit-mooncake))

## Installation
`lit install gerrustalker/steam-openidc`
### Manually
1. Clone this repository to your `deps` folder
2. Run `lit install`
